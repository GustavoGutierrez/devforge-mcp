Getting Started

DevForge ships three binaries: devforge (CLI/TUI), devforge-mcp (MCP stdio server), and dpf (media runtime).

Install

brew install GustavoGutierrez/devforge/devforge

Or from source:

git clone https://github.com/GustavoGutierrez/devforge.git
cd devforge
go build ./...

Requirements

  • Go 1.24+ for source builds.
  • FFmpeg in $PATH for video/audio tools.
  • Gemini API key only for Gemini-backed tools.

Configuration

{
  "gemini_api_key": "",
  "image_model": "gemini-2.5-flash-image"
}

Default path: ~/.config/devforge/config.json. Override with DEV_FORGE_CONFIG.
