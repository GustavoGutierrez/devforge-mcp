devforge_markdown_to_pdf (markdown_to_pdf)

Converts Markdown into production-grade PDF documents for PRPs, specs, reports, invoices, and deliverables.

Input modes (choose one)

  • input: file path to a markdown file.
  • markdown_text: inline UTF-8 markdown string.
  • markdown_base64: base64-encoded UTF-8 markdown.

Output modes

  • output: explicit file path (single output file).
  • output_dir + file_name: write to directory with chosen filename.
  • inline: true: return base64 PDF bytes in response.

Layout and page options

  • page_size: a4 | letter | legal.
  • page_width_mm and page_height_mm: custom dimensions.
  • layout_mode: paged or single_page.

Theme options

  • theme: invoice | scientific_article | professional | engineering | informational.
  • theme_config: raw advanced override object forwarded to dpf.
  • theme_override typed fields: name, body_font_size_pt, code_font_size_pt, heading_scale, margin_mm.

Practical examples

PRP or technical spec from file

{
  "tool": "devforge_markdown_to_pdf",
  "input": "PRPs/002-feature.md",
  "output": "dist/prp.pdf",
  "theme": "engineering",
  "page_size": "a4"
}

Inline report to base64 response

{
  "tool": "devforge_markdown_to_pdf",
  "markdown_text": "# Incident Report\n\n## Summary\n- Resolved.",
  "inline": true,
  "theme": "professional"
}

Invoice-style PDF with themed overrides

{
  "tool": "devforge_markdown_to_pdf",
  "input": "docs-data/invoice.md",
  "output_dir": "dist/pdf",
  "file_name": "invoice-042.pdf",
  "theme": "invoice",
  "theme_override": {
    "name": "Invoice Tight",
    "body_font_size_pt": 10,
    "code_font_size_pt": 9,
    "heading_scale": 1.08,
    "margin_mm": 12
  }
}

Base64 source for generated markdown

{
  "tool": "devforge_markdown_to_pdf",
  "markdown_base64": "IyBUaXRsZQoKQ29udGVudC4=",
  "output": "dist/generated.pdf",
  "layout_mode": "single_page",
  "theme_config": {
    "code_block_background": "#F6F8FA"
  }
}
