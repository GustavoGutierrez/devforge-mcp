# DevForge Mascot Prompt

**"Forge your design system — from pixel to production."**

Use this prompt with **Google Nanobanana** (or any AI image generator) to create a mascot for the DevForge MCP project.

---

## Prompt

```
A friendly, modern mascot character for a developer productivity tool called "DevForge".
The character is a small robotic blacksmith dwarf or gnome wearing a dark hoodie and
work goggles. It holds a glowing digital hammer made of light/code, actively
forging a glowing pixel gemstone on an anvil made of circuit board traces. The
scene is set in a cozy dimly-lit workshop with floating code snippets, design
tokens, and UI wireframe fragments in the background. The style is clean vector
illustration with soft gradients, flat design with subtle depth, vibrant accent
colors: electric indigo primary, warm amber highlights, cool teal for the code
elements. Friendly expression, approachable and techy. Transparent background,
centered composition, square format, highly detailed, professional quality
```

---

## Style Reference

| Attribute | Value |
|---|---|
| **Style** | Flat vector + subtle gradients, clean lines |
| **Palette** | Indigo `#6366f1`, Amber `#f59e0b`, Teal `#14b8a6`, Dark `#1e1e2e` |
| **Character** | Small robotic gnome/dwarf blacksmith |
| **Props** | Glowing code-hammer, circuit-board anvil, floating UI snippets |
| **Mood** | Cozy, approachable, productive |
| **Background** | Transparent |
| **Format** | Square, centered |
